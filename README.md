
## About

This starter template includes:
- Vite + Solid JS + Solid Router
- Typescript
- Tailwind CSS


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Start the app

```bash
  yarn dev
```
